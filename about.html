Welcome to CovStat!

CovStat is an intuitive visual representation of live COVID-19 data based on the COVID Tracking API. It was built by Safin Singh using HTML/CSS, JavaScript, and Chart.JS. If you're a developer, feel free to check out the CovStat API, which fetches and graphs data.
